# Goodfellas

Tema para VSCode com estética de máfia italiana — tons de camo verde, âmbar e pergaminho.

## Variantes

- **Goodfellas Dark** — fundo escuro com acentos em verde camo
- **Goodfellas Dark Ember** — fundo escuro com acentos quentes em âmbar
- **Goodfellas Light** — fundo pergaminho com texto em oliva escuro
